# ScientificResources
Scientific Resources System Management

## Entities
- Researcher
- Reviewer
- User

## Functionalities
- Authentification
- Researcher can publish an article
- Reviewer can review an article
  - Rate an article  
  - Approve an article
- User can display approved articles

## Technologies
- JEE (Servlet/JSP)
- Apache Tomcat 7
- MySQL
